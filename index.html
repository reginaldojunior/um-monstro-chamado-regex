Conceitos Básicos de Linguagem

Linguagem é definida por seus simbolos, cadeias e a linguagem propriamente dita. Para reconhecemos uma linguagem precisamos de dois conceitos a gramática da linguagem e seus reconhecedores.

A Gramática

Faz a análise na especificação formal da linguagem, considerando todas as regras que o torna uma linguagem.

Reconhecedores

São os modelos que será dados para as análises lexicas e sintaticas para os compiladores.

Os simbolos da linguagem.

Os simbolos em uma linguagem também conhecido como átomos ou palavras são as representações empregadas para as formações de cadeias.

As cadeias da linguagem.

As cadeias são formadas através dos simbolos, em um conjunto finito de dados.

Alfabetos

Diferente da cadeia que é formada por a concatenação dos simbolos formando um conjunto finito não vazio. O alfabeto é um conjuto de simbolos.

Exemplo 1.0

Dado o alfabeto:
A = {0, 1}
Podemos formar as seguintes cadeias:
0, 1, 01, 10, 110, 001...

Exemplo 1.1

Dado o alfabeto:
A = {a, b, c, d, ..., w, y, x, z}
Podemos formar as seguintes cadeias:
casa, carro, programacao, meetup, backendbrasil, comunidade...
